Cobblestone

This is Cobblestone, it's a notetaking app based on EyeScary Development's Harmony code editor, it features the same great experience, and just replaces the syntax highlighting with markdown interpreted by the Lead markdown interpreter

Why is it called cobblestone?

because lava and water make cobblestone and obsidian, and cobblestone is worse than obsidian in every way

How get?

just get EyeScary Development's AGPM package manager and you can install it with that, the command is cobble btw when you install it
